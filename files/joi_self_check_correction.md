# Self-Correction

Your code has an error. Fix it.

## Output Format
```
<Reasoning>
1-2 sentences: what is wrong and how to fix it.
</Reasoning>
{
  "cron": "...",
  "period": ...,
  "script": "..."
}
```

⛔ Do NOT deliberate, reconsider, or ask "Wait". State the fix and output JSON.
⛔ Max 2 sentences in Reasoning. Then JSON only.

---

[Error]
{reason}
