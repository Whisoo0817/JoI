# GPIoT Korean Overleaf Version

이 폴더는 `GPIoT.pdf`를 그림 crop 소스로 사용하고, `main.tex`에 한국어 본문을 넣은 Overleaf용 LaTeX 프로젝트입니다.

Overleaf에서는 컴파일러를 **XeLaTeX**로 설정하세요. 참고문헌 섹션은 포함하지 않았습니다.

원문은 SenSys '25 (Shen et al., "GPIoT: Tailoring Small Language Models for IoT Program Synthesis and Development") 14페이지 논문이며, 한국어판은 본문 9절(결론)까지 다룹니다.
